La configuración del archivo de prometheus para la visualización de las métricas tiene disponible dos modos: un primer modo donde se puede usar directamente el servicio de node exporter y un segundo modo que se puede usar con IPs de uno o varios equipos.

Para añadir las fuentes de datos de Grafana:
1. Accede a Grafana: http://localhost:3000 (usuario y contraseña especificados en el docker-compose)
2. Ve a Configuración > Data Sources > Add data source
3. Selecciona Prometheus en la lista
4. En el campo URL escribe http://IP:9090 donde IP corresponde con la IP del equipo donde esté corriendo el servicio de prometheus
5. Hacer clic en Save & Test

A la hora de usar Grafana, las credenciales se le pueden especificar en el archivo docker-compose.yml. Como Dashboard, sugiero usar el de ID 1860. Para usarlo, si no está creado:
1. Accede a Grafana: http://localhost:3000 (usuario y contraseña especificados en el docker-compose)
2. Ve a Dashboard > Import
3. Introduce el ID 1860 y haz clic en Load
4. Selecciona tu fuente de datos prometheus
5. Haz clic en Import


Recordatorio: Hay que habilitar los puertos 9100 para Node Exporter, el 9417 para Docker Exporter y el 9090 para Prometheus
